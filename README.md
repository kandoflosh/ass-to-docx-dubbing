# Project Title

This project is a tool for dubbing videos into formatted Word documents (docx), ensuring seamless synchronization of subtitles and audio.

## Installation Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/kandoflosh/ass-to-docx-dubbing.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ass-to-docx-dubbing
   ```
3. Install required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:
   ```bash
   python app.py
   ```
